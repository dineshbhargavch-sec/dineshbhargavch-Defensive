# Phishing Simulation & Detection Lab Using Gophish (2025)

## Overview
This repository contains artifacts from a controlled phishing simulation environment using **Gophish** to demonstrate common phishing techniques, analyze user interaction patterns, and surface behavioral indicators for security training.

## Key Highlights
- Deployed Gophish in a controlled lab environment.
- Created realistic phishing email templates and landing pages.
- Analyzed user responses and interaction metrics.
- Documented phishing tactics, prevention strategies, and awareness training recommendations.

## Folder Structure
- `Gophish-Templates/` – phishing email & landing page templates.
- `Simulation-Reports/` – campaign results and interaction data.
- `Screenshots/` – visual evidence of phishing dashboard and results.

## Skills Demonstrated
- Phishing attack simulation & analysis
- Security awareness assessment
- Reporting and documentation
