# SOC Alert Monitoring & Incident Analysis Project (2025)

## Overview
Configured a SOC monitoring environment using **Splunk** to ingest logs, detect threats, and analyze security incidents in real time.

## Key Highlights
- Ingested system, network, and endpoint logs.
- Created alert searches for brute‑force attempts, phishing, and suspicious activity.
- Performed first‑level investigation and triage of triggered alerts.
- Produced SOC‑style incident reports and trend summaries.

## Folder Structure
- `Splunk-Configs/` – saved search & dashboard configs.
- `Reports/` – example incident analysis reports.
- `Screenshots/` – Splunk dashboards & alerts.

## Skills Demonstrated
- SOC monitoring & alerting
- Log analysis & threat detection
- Incident reporting
