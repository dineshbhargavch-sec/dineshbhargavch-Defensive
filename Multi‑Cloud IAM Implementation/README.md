# Multi‑Cloud Identity & Access Management (AWS & Azure) 2025

## Overview
This project demonstrates secure IAM configuration across AWS and Azure, enforcing least‑privilege access, MFA, RBAC, and custom access policies.

## Key Highlights
- Configured AWS IAM users, roles, and policies.
- Configured Azure AD roles, groups, and access assignments.
- Applied strict password policies and Multi‑Factor Authentication (MFA).
- Created custom IAM policies for S3, EC2, Azure Storage, and Azure VMs.

## Folder Structure
- `IAM-Policies/` – JSON policy files for AWS & Azure.
- `Documentation/` – explanation of policy logic.
- `Screenshots/` – console configurations.

## Skills Demonstrated
- Cloud security & IAM
- Policy authoring
- Multi‑cloud access control
